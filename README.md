### generator-dtcli 脚手架使用
```

   sudo npm install -g yo (已安装yeoman，可以忽略此步)   

```
* 新建目录
```

   mkdir projectName && cd projectName                  

```
* 运行脚本命令
```

   sudo npm install generator-dtcli -g                  

```
* 查看本地脚手架  

```

   yo --generators                                          

```
* 新建本地脚手架

```

   yo 脚手架名 （根据提示生成脚手架)                                      

```
### 使用技巧 
  ```
  npm run getScafflod -- [pc ,mobile ,apollo]                         
  选择要安装的脚手架，目前支持 pc,mobile,apollo ;                                            
  ```
