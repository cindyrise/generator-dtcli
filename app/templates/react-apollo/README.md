#### 安装教程

1、 安装依赖包。
```
npm install 或者cnpm install 或者yarn(推荐)
```

2、运行脚手架。
 ```
 npm start
 ```

3、将会开启8080端口.
```
http://127.0.0.1:8090
```

4、打包发布: 默认打包后的文件统一放到dist文件夹下  

```
npm run build
```
